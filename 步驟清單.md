1.輸入下面指令，清理GCP
cd ~
rm -rf (這裡填GCP@後面的)
2.下載github專案
git clone [https://github.com/你的帳號/你的專案名稱.git]
3.進入專案資料夾
cd 你的專案資料夾
4.下載Node.js
安裝套件
npm install
編譯專案
npm run build
5.在本地資料夾跟目錄創建deploy.bat(一鍵更新專案到github)，並輸入
@echo off
echo [1/3] Adding changes...
git add .
set /p commit_msg="Enter commit message: "
git commit -m "%commit_msg%"
echo [2/3] Pushing to GitHub...
git push
pause
6.回到雲端新增一個資料夾 update.sh(一鍵更新雲端)
nano update.sh
7.在update.sh資料夾輸入下面指令
#!/bin/bash
echo "Updating Project..."
cd ~/my-personal-website
<!--(我的~/GCP-4/personal-portfolio)-->
git pull
npm install
npm run build
sudo systemctl restart nginx
echo "Update Complete!"
8.給予update.sh執行權限
chmod +x update.sh
9.下載nginx
更新套件清單
sudo apt update
安裝 Nginx
sudo apt install nginx -y
確認是否安裝成功
sudo nano /etc/nginx/sites-available/default
10.在default檔案輸入下面指令，記得把路徑改成你自己的(GCP@前面的)
server {
    listen 80 default_server;
    server_name _;

    # 告訴 Nginx 網站打包後的靜態檔案在哪裡(路徑要看個人專案)
 <!--(我的root /home/ereffss8888/GCP-4/personal-portfolio/dist;) -->
    root /home/你的帳號/my-personal-website/dist;
    index index.html;

    # 這是 React 單頁應用 (SPA) 不會報錯的必備設定
    location / {
        try_files $uri $uri/ /index.html;
    }
}
11.重新啟動nginx
確認是否成功(顯示 syntax is ok)
sudo nginx -t
重啟nginx
sudo systemctl restart nginx
12.最終指令
在本地更新就輸入，或是透過資料夾開啟deploy.bat
.\deploy.bat
在雲端更新就輸入
./update.sh
